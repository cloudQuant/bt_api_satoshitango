# bt_api_satoshitango

SatoshiTango exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_satoshitango
```

## Usage

```python
from bt_api_satoshitango import SatoshiTangoExchangeDataSpot, SatoshiTangoRequestDataSpot
```

## Features

- REST API support
- Bearer token authentication
- Symbol format: `BTC/ARS` etc.
